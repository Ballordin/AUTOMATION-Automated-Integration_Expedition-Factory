# HMI Design (NB Designer)

Interface designed for the **Omron NB7W-TW01B** touch screen.

## Key Screens
1. **Main Dashboard:** Real-time visualization of the production line status.
2. **Manual Control:** Allows individual testing of actuators and conveyors.
3. **Expedition Manager:** A dedicated interface where the operator selects the part color and quantity to be retrieved from the warehouse.
4. **Alarms/Diagnostics:** Displays system errors and emergency stop status.

> **Software Requirement:** Omron NB Designer.
